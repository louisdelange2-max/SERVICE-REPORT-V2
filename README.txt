# Service Report PWA

Upload these files to a GitHub repository and enable GitHub Pages.

Files included:
- index.html
- manifest.webmanifest
- sw.js
- icons/icon-192.png
- icons/icon-512.png

After publishing:
1. Open the site in Chrome on Android.
2. Use the browser menu and choose Install app / Add to Home screen.
3. In the app, save your Google Apps Script Web App URL under Connect and press Sync.
